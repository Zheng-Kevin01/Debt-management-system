# 債務管理系統 By Z.W.Y.

[ZH Taiwan] [EN](README-EN.md)

一款方便的記帳程式,協助您管理日常生活中的各種債務,計算風險係數,快速辨識哪些債務優先順序最高！

# 前置作業

請先Python安裝套件

!pip install pandas matplotlib openpyxl pyinstaller

# 程式運轉

1. 在桌面建立資料夾：C:\Users\user\Desktop\NAME

2. 執行程式,將出現UI介面,輸入您的債務資料,系統將幫你計算每筆債務的風險係數,輸出成 xisx 文件,並輸出圖表
