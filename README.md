# 債務管理系統 By Z.W.Y 
[ZH Taiwan] [EN](README-EN.md)

--最新版本V1.5.0-- //本地端應用

一個方便你在日常生活中整理各種債務的記帳本程式,幫你計算風險係數
讓你快速知道哪筆債務為最優先級!

# 前置作業

請先安裝可運行Python之開發環境
安裝套件 !pip install pandas matplotlib openpyxl pyinstaller

# 程式運轉

1.於桌面創建一個資料夾C:\Users\user\Desktop\NAME

2.執行程式碼,跳出GUI介面,輸入債務資料由係統協助您將各債務的風險係數算出,並彙整成.xisx檔,以及輸出債務圖表~
