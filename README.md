# Debt Management System By Z.W.Y

--Latest version V1.5.0-- //Local application

A convenient accounting program for managing various debts in your daily life, helping you calculate risk coefficients and quickly identify which debt has the highest priority!

# Prerequisites

Please install a Python development environment first.
Install packages: !pip install pandas matplotlib openpyxl pyinstaller

# Program Operation

1. Create a folder on your desktop: C:\Users\user\Desktop\NAME

2. Execute the program code. A GUI interface will appear. Enter your debt information, and the system will calculate the risk coefficient for each debt, compile it into a .xisx file, and output a debt chart.
