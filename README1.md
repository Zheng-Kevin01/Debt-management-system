# 💰 債務管理系統（Debt Management System）

一套以 **Python + Tkinter + Pandas** 打造的桌面型債務管理與風險分析系統，  
能協助使用者清楚掌握債務狀況、風險排序、可運用資金，並產生視覺化圖表與 Excel 報表。

> 📌 本專案特別針對「**到期日 × 金額 × 利率**」進行風險評估，  
> 讓債務不再只是數字，而是有**優先處理順序**的戰術目標。

---

## ✨ 功能特色

### 📋 債務管理
- 新增 / 修改 / 刪除債務
- 支援多筆債務資料
- 債務資料自動保存（`debt_data.xlsx`）

### ⚠️ 風險指數計算
- 依據以下因子計算 **風險指數**
  - 債務金額
  - 利率
  - 距離到期日天數
- 自動分級：
  - 🚩 高風險
  - ⚠️ 中風險
  - ✅ 低風險
- 系統畫面依「風險指數由高到低」排序

### 🎯 視覺化標示
- 三色旗標（紅 / 黃 / 綠）
- Treeview 背景色直接顯示風險等級
- 金額、風險指數皆使用 **千分位格式**，一眼就懂

### 💵 收入與可運用資金分析
- 月收入可輸入並**永久保存**
- 自動計算：可運用資金 = 月收入 - 債務總額
- 顏色提示：
- 🟢 正值
- 🔴 負值

### 📊 風險分析報告
- 一鍵顯示：
- 高 / 中 / 低 風險債務清單
- 方便快速判斷「先處理誰」

### 📁 報表與圖表輸出
- 產生 Excel 報表（`債務報表.xlsx`）
- 含風險排序
- 含 TAG 標示
- 含總金額 / 收入 / 產生日期
- 輸出圖表：
- 📈 風險指數排序圖
- 📊 債務金額分布圖

---

## 🧠 風險指數計算邏輯（核心）
風險指數 = 金額 × 利率 × 到期緊急因子

- 到期日越近 → 緊急因子越高
- 已銷帳債務 → 風險指數 = 0

這代表：
> 💡 **利率一樣時，快到期的債務會被優先拉高風險**
